# BookStoreWeb-MVC

## Features:
- CRUD Operation using MVC (Shopping Cart)
- Repository Pattern
- Identity .Net Core and User Management (Admin, Customer, Company)
- Order Management wth Stripe
- Email confirmation
- Deployment to Azure
