# BookStoreWeb-MVC
Platform: .Net Core 6

## Features:
- CRUD Operation using MVC (Shopping Cart)
- Repository Pattern
- Identity .Net Core with Email confirmation 
- User Management (Admin, Customer, Company)
- Order Management wth Stripe
- Deployment to Azure
